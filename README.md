# Agentic-Systems-in-Biomedicine

Langflow-based agent prototypes for biomedical research and interpretation workflows.

## Repository guide

- `Agents/README.md` explains how the agent folder is organized and why the flows use a custom OpenAI model component.
- `Agents/LabValuesInterpreterAgent/README.md` documents the laboratory interpretation flow.
- `Agents/DeepResearchAgent/README.md` documents the multi-agent deep research flow.
- `Agents/PersonalAssistantAgent/README.md` documents the multi-tool personal assistant flow.
- `Agents/StatisticalAssistantAgent/README.md` documents the Python-based statistical assistant flow.
- `Agents/HPCOperatorAgent/README.md` documents the Chimera-focused HPC operator flow.
- `Agents/LiteratureAgent/README.md` documents the literature-search and synthesis flow.
