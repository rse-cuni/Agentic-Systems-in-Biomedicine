# Agentic-Systems-in-Biomedicine

Langflow-based agent prototypes for biomedical research and interpretation workflows.

## Repository guide

- `Agents/README.md` explains how the agent folder is organized and why the flows use a custom OpenAI model component.
- `Agents/LabValuesInterpreter/README.md` documents the laboratory interpretation flow.
- `Agents/DeepResearchAgent/README.md` documents the multi-agent deep research flow.
- `Agents/PersonalAssistant/README.md` documents the multi-tool personal assistant flow.
- `Agents/StatisticalAssistant/README.md` documents the Python-based statistical assistant flow.
- `Agents/HPCOperator/README.md` documents the Chimera-focused HPC operator flow.
