# Methods Design

Use this skill for experimental design, statistical planning, and methods critique.

## Design Checklist

- State the target claim and what evidence could support it.
- Define controls, baselines, negative checks, and sensitivity checks.
- Identify confounders, leakage risks, and invalid comparison risks.
- Predefine metrics, thresholds, and stopping criteria.
- Separate pilot feasibility from final inference.

## Analysis Planning

- Specify required inputs and minimum data quality.
- State assumptions and what happens if they fail.
- Include runtime or sample-size considerations when relevant.
- Define what result would change the PI decision.

## Critique Focus

When reviewing, look for missing controls, mismatched metrics, underpowered comparisons, multiple testing, leakage, unclear thresholds, and overinterpretation.
